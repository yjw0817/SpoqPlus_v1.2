아이콘 저장 방법

  1. 각 아이콘 영역을 마우스 오른쪽 클릭
  2. "이미지로 저장" 선택
  3. 192x192 아이콘은 icon-192.png로 저장
  4. 512x512 아이콘은 icon-512.png로 저장
  5. 저장 위치: /public/images/ 폴더

또는 개발자 도구 사용:

  1. F12를 눌러 개발자 도구 열기
  2. Console 탭에서 아래 명령어 실행
  3. 자동으로 다운로드됨
